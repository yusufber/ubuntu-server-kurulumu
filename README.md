# ubuntu-server-kurulumu
Bu kılavuzda Ubuntu 22.04 Sunucu kurulum adımları açıklanmaktadır.

## Temel Linux Komutları

### Süper Kullanıcı Moduna Geçme
``` ruby
sudo -i
```
### Sistemi yeniden başlatma
``` ruby
sudo reboot
```

### Sistemi kapatma
``` ruby
sudo shutdown -h now
```

### Ip adresini öğrenme
``` ruby
ip addr
```

### Disk boyutunu Öğrenme
``` ruby
df -h
```
